## Page-Level SEO Rules

### Every page must have:
- Unique `<title>` tag (max 60 chars, includes "Praha 3" or "Žižkov" where natural)
- Unique `<meta description>` (max 160 chars, includes CTA language)
- `<h1>` containing primary keyword for that page
- Open Graph tags (og:title, og:description, og:image)
- Canonical URL
- Structured data (BarberShop for homepage, Article for blog, Service for services)

### Blog post SEO template:
```
Title: [Primary Keyword] — [Benefit/Hook] | Gee & Geesus
Description: [Excerpt with keyword + CTA]. Barbershop Praha 3, Žižkov.
H1: [Title without brand suffix]
URL: /blog/[keyword-slug]
```

## Internal Linking Rules

1. **Blog → Services**: Every grooming/care article must link to at least one service on `/sluzby`
2. **Blog → Products**: Product-related articles must link to `/produkty`
3. **Blog → Booking**: Every blog post ends with a booking CTA section linking to `/rezervace`
4. **Services → Booking**: Each service card links to `/rezervace`
5. **Homepage → All sections**: Homepage teasers link to their full pages
6. **Footer**: All main pages linked in footer nav
7. **Breadcrumbs**: All non-homepage pages show breadcrumb navigation

## Technical SEO

- **Sitemap**: Dynamic `/sitemap.xml` generated by Next.js (all pages + blog posts)
- **Robots.txt**: Allow all, disallow /profil/*, /manage.html
- **Image optimization**: Next.js Image component, WebP automatic, lazy loading
- **Core Web Vitals**: Target LCP < 2.5s, CLS < 0.1, INP < 200ms
- **Mobile-first**: All pages designed mobile-first, tested at 375px
- **Page speed**: Static generation (SSG) for all public pages, ISR for blog
- **hreflang**: cs-CZ primary, en-GB planned for future
